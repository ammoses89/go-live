API

Check Status Endpoint: /api/[distributor]
params:
  • upc: Required for Spotify/iTunes/Rdio
  • artist: Required for Amazon/Deezer/Google
  • album_title: Required for Amazon/Deezer/Google

Outlets:
  • spotify
  • itunes
  • rdio
  • amazon
  • deezer
  • google

example: http://is-it-live.appspot.com/api/spotify?upc=884502232769
http://is-it-live.appspot.com/api/deezer?album_title=You+Forget+It+In+People&artist=Broken+Social+Scene
